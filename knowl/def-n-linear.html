Definition 10.1. Multilinearität.

  1. Eine Abbildung \(f: V^n \rightarrow W\) heißt \(n\)-linear (multilinear), wenn gilt:
    \begin{equation*} \begin{split} &:\iff f \text{ ist linear in jedem Faktor von $V^n$}\\ &:\iff \text{Für jedes $j\in \set{1,\dots, n}, \forall (v_1, \dots, v_{n-1}) \in V^{n-1}$ gilt:}\\ &V\rightarrow W: v\mapsto f(v_1, \dots, v_{j-1, \underbrace{v}_{\text{Stelle } j}, v_j, \dots, v_{n-1}}) \text{ ist linear.} \end{split} \end{equation*}
  2. Gilt in (a) \(W = K\text{,}\) so heißt \(f\) \(n\)-Linearform.
  3. \(\Lin_n(V,W) :=\) Menge aller \(n\)-linearen Abbildungen \(V^n \rightarrow W\text{.}\)
  4. \(f\in \Lin_n(V,W)\) heißt alternierend \(:\iff \forall (v_1, \dots, v_n)\in V^n, \exists 1\leq i < j \leq n\) mit \(v_i = v_j\text{,}\) so gilt \(f(v_1, \dots, v_n) = 0\text{.}\)
  5. \(\Alt_n(V,W) := \set{f\in \Lin_n(V,W) \mid f \text{ ist alternierend}}\text{.}\)
Bemerkung: \(\Lin_1 (V,W) = \Lin(V,W)\text{.}\)
