Definition 10.3. Charakteristik.

Die Charakteristik von \(K\) ist definiert als
\begin{equation*} \operatorname{char} K = \begin{cases} \min \set{n\in \N \mid n\cdot 1_k = 0_k}, & \text{falls } \exists n\in \N \text{ mit } n\cdot 1_k = 0_k\\ 0, &\text{sonst} \end{cases} \end{equation*}
\(\char(K)\) ist \(0\) oder eine Primzahl.
